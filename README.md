# 電商行銷自動化系統 使用說明

## 系統概述

電商行銷自動化系統是一個運用 AI 技術的一站式行銷工具，幫助電商賣家快速產生各種行銷素材。

## 核心功能

### ① AI 商品文案自動生成

**功能說明：**
- 自動生成 5 種不同風格的商品文案
- 支援網址自動分析
- 可編輯、儲存模板、一鍵複製
- 可匯出 CSV 批次上架

**文案類型：**
1. **SEO 版** - 優化搜尋引擎排名
2. **電商銷售版** - 強調特色與優惠
3. **感性故事版** - 情感共鳴
4. **短標題** - 簡潔有力
5. **蝦皮規格+賣點** - 蝦皮平台專用格式

**使用流程：**
1. 輸入商品名稱
2. （選填）輸入商品網址，點擊「AI 分析」自動抓取商品資訊
3. （選填）補充商品描述
4. 選擇要產出的文案類型（可複選）
5. 點擊「開始生成文案」
6. 查看結果，可進行編輯、複製或存為模板
7. 點擊「匯出 CSV」下載批次上架檔案

---

### ② AI 批次產圖

**功能說明：**
- 批次生成 3-20 張商品情境圖
- 支援多種風格與背景替換
- 自動去背功能
- 可加入品牌 Logo
- 一鍵打包下載 ZIP

**風格選項：**
- IG 風格 - 適合社群媒體
- 清新少女 - 柔和清新風格
- 男性科技感 - 專業科技風
- 日本簡約 - 無印良品風

**背景選項：**
- 純白背景
- 木桌
- 大理石
- 廚房
- 戶外
- 浴室

**使用流程：**
1. 上傳商品圖片
2. 選擇風格模板
3. 選擇背景類型
4. 調整生成數量（3-20 張）
5. 選擇是否自動去背、加入 Logo
6. 點擊「開始生成圖片」
7. 選擇需要的圖片
8. 點擊「打包下載 ZIP」

---

### ③ 影音腳本 AI 生成

**功能說明：**
- 生成 TikTok、Reels 短影音腳本
- 包含分鏡表、拍攝建議
- 轉場效果建議
- 鏡位與配樂建議
- 可匯出拍攝清單

**影片風格：**
- **帶貨口播** - 主持人介紹產品特色
- **商品展示** - 360度產品展示
- **情境故事** - 用故事呈現產品

**使用流程：**
1. 輸入商品名稱
2. 輸入商品描述
3. 選擇影片風格
4. 設定影片長度（10-60 秒）
5. 點擊「開始生成腳本」
6. 查看分鏡表與拍攝建議
7. 點擊「匯出拍攝清單」下載文字檔

**腳本內容包含：**
- 每個場景的畫面描述
- 旁白文案
- 鏡位建議
- 所需道具
- 轉場效果
- 配樂風格
- CTA 文案

---

### ④ 多平台格式轉換

**功能說明：**
- 自動轉換為各電商平台格式
- 符合平台字數與規格限制
- 一鍵匯出可上架的 CSV

**支援平台：**

| 平台 | 標題字數限制 | 圖片尺寸 | 特色功能 |
|------|-------------|---------|---------|
| 蝦皮 Shopee | 60 字 | 1200x1200 | 規格列表、賣點整理、CSV 匯出 |
| Momo 購物 | 50 字 | 800x800 | 分段描述、特色點整理 |

**使用流程：**
1. 輸入商品名稱、價格
2. 輸入商品描述
3. 選擇目標平台
4. 點擊「開始轉換」
5. 查看轉換後的格式
6. 點擊「下載 CSV」（蝦皮專用）

**蝦皮格式包含：**
- 標題（60字內）
- 商品描述
- 規格列表
- 賣點整理
- CSV 批次上架檔

**Momo 格式包含：**
- 標題（50字內）
- 分段式描述
- 規格表

---

### ⑤ FB 廣告素材自動生成

**功能說明：**
- 依據產品類型自動生成廣告文案
- 符合 Facebook 廣告規格
- 一次產出 1-10 篇廣告
- 即時驗證格式正確性
- 模擬 FB 廣告預覽

**使用流程：**
1. 輸入商品名稱、價格
2. 輸入商品描述
3. 選擇生成數量（1-10 篇）
4. 點擊「開始生成廣告」
5. 查看 FB 廣告預覽
6. 檢查格式驗證結果
7. 點擊「複製」使用文案

**廣告內容包含：**
- 主要文字（建議 125 字內）
- 標題（40 字以內）
- 說明（30 字以內）
- CTA 按鈕文字
- 格式驗證與錯誤提示

**FB 廣告規格：**
- 圖片尺寸：1200 x 628 px（比例 1.91:1）
- 圖片中文字不超過 20%
- 主要文字建議 125 字內完整顯示
- 標題最多 40 字
- 說明最多 30 字

### ⑥ 受眾分析

**功能說明：**
- 自動分析產品可能的受眾群體
- 提供人口統計資訊
- 推薦投放平台
- 建議關鍵字與目標市場

**使用方式：**
1. 輸入商品名稱，點擊「分析」
2. 或輸入商品網址，點擊「AI 分析」

**分析結果包含：**

**建議受眾群體：**
- 受眾名稱與描述
- 市場規模（小眾/中型/大眾）
- 相關性分數（0-100）
- 建議投放平台

**人口統計分析：**
- 年齡範圍
- 性別分布
- 興趣標籤
- 行為特徵

**其他資訊：**
- 關鍵字建議
- 目標市場（國家/地區）

**範例：**
```
商品：自拍棒

建議受眾：
1. 旅遊愛好者 (相關性 95%)
   - 市場規模：大眾市場
   - 建議平台：Instagram, Facebook

2. 社群內容創作者 (相關性 90%)
   - 市場規模：中型市場
   - 建議平台：Instagram, Facebook

年齡範圍：18-24, 25-34, 35-44
性別：女性 65%, 男性 35%
興趣：旅遊、攝影、社群媒體、時尚、美食
```

## 技術架構

### 前端技術
- React + TypeScript
- Tailwind CSS
- Vite

### AI 整合
- 支援 GPT-4、Claude 等商業 AI 模型
- 可透過 API 連接 OpenAI、Anthropic
- 網頁分析與內容抓取

```
src/
├── types/
│   └── marketing.ts          # 型別定義
├── services/
│   └── aiMarketingService.ts # AI 服務層
├── components/
│   └── marketing/
│       ├── MarketingAutomationSystem.tsx  # 主系統
│       ├── CopywritingGenerator.tsx       # 文案生成
```

## 使用提示

### 文案生成
- 提供越詳細的商品描述，生成的文案越精準
- 網址分析功能可自動抓取商品資訊，節省輸入時間
- 可先生成多種類型，再挑選最適合的使用

### 圖片生成
- 上傳的商品圖片建議清晰、主體明確
- 去背功能適合需要合成到不同背景的情境
- 批次生成後可選擇性下載需要的圖片

### 影片腳本
- 15 秒適合 TikTok、Reels
- 30-60 秒適合 YouTube Shorts
- 拍攝清單包含完整的道具、鏡位建議

### 平台轉換
- 注意各平台的字數限制
- 蝦皮 CSV 可直接用於批次上架
- Momo 需要分段描述，系統會自動處理

### FB 廣告
- 系統會自動驗證格式
- 主要文字超過 125 字會被截斷
- 建議搭配高品質商品圖片

### 受眾分析
- 分析結果可用於設定廣告受眾
- 相關性分數越高，轉換率通常越好
- 關鍵字可用於 SEO 與廣告投放

## 常見問題

**Q: AI 生成的內容準確嗎？**
A: AI 生成的內容僅供參考，建議根據實際情況進行調整和優化。

**Q: 可以使用自己的 AI API 嗎？**
A: 可以，請在 `.env` 檔案中設定您的 API Key。

**Q: 圖片生成需要多久？**
A: 視生成數量而定，通常 3-5 張約需 3-5 秒（模擬）。實際接入 AI 圖片生成服務後，可能需要 30-60 秒。

**Q: 支援其他電商平台嗎？**
A: 目前支援蝦皮和 Momo，未來會新增更多平台。

**Q: 可以批次處理多個商品嗎？**
A: 目前是單一商品處理，批次功能規劃中。
