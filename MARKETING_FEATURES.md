# 電商行銷自動化系統 - 功能總覽

## 🎯 系統定位

一個運用 AI 技術的一站式電商行銷工具，幫助賣家快速產生各種行銷素材，大幅提升工作效率。

---

## ✨ 六大核心功能

### 1️⃣ AI 商品文案自動生成

**解決的問題：**
- 不知道怎麼寫吸引人的商品文案
- 不同平台需要不同風格的文案
- 手動撰寫耗時費力

**功能特色：**
- 一鍵生成 5 種風格文案
- 網址自動分析商品資訊
- 可編輯、複製、存模板
- 匯出 CSV 批次上架

**5 種文案類型：**

| 類型 | 用途 | 特色 |
|------|------|------|
| SEO 版 | 搜尋引擎優化 | 關鍵字密度高、結構完整 |
| 電商銷售版 | 電商平台銷售 | 強調優惠、賣點、CTA |
| 感性故事版 | 品牌形象建立 | 情感共鳴、故事性強 |
| 短標題 | 廣告投放 | 簡潔有力、吸睛 |
| 蝦皮規格+賣點 | 蝦皮平台 | 符合蝦皮格式、規格清晰 |

**範例輸出：**
```
商品：智能藍牙自拍棒

【SEO 版】
標題：【2024最新】智能藍牙自拍棒 | 專業級自拍神器 | 藍牙遙控 | 台灣現貨

內容：智能藍牙自拍棒是您旅遊自拍的最佳選擇！採用航太級鋁合金材質，
輕巧便攜卻堅固耐用。內建藍牙 5.0 遙控器，10 公尺遠距離操控，
支援 iOS 與 Android 雙系統...

關鍵字：自拍棒、藍牙自拍棒、旅遊必備、直播器材、vlog設備
```

### 2️⃣ AI 批次產圖

**解決的問題：**
- 商品照片單調、缺乏情境感
- 需要大量不同背景的商品圖
- 專業攝影成本高昂

**功能特色：**
- 批次生成 3-20 張圖片
- 4 種風格模板
- 6 種背景選擇
- 自動去背功能
- 可加入品牌 Logo
- ZIP 打包下載

**風格模板：**

| 風格 | 適用場景 | 特色 |
|------|---------|------|
| IG 風格 | 社群媒體 | 明亮、高對比 |
| 清新少女 | 女性商品 | 柔和、粉嫩色調 |
| 男性科技感 | 3C 產品 | 冷色調、專業感 |
| 日本簡約 | 生活用品 | 極簡、無印風 |

**背景選項：**
- 純白背景（商品目錄）
- 木桌（溫馨感）
- 大理石（高級感）
- 廚房（廚具類）
- 戶外（運動用品）
- 浴室（美妝保養）

**流程：**
```
上傳商品圖 → 選擇風格 → 選擇背景 → 設定數量 → AI 生成 → 打包下載
```

### 3️⃣ 影音腳本 AI 生成

**解決的問題：**
- 不知道如何規劃短影音內容
- 缺乏專業的拍攝經驗
- 分鏡、鏡位規劃困難

**功能特色：**
- 15-60 秒短影音腳本
- 完整分鏡表
- 拍攝建議（鏡位、道具）
- 轉場效果建議
- 配樂風格建議
- CTA 呼叫設計
- 匯出拍攝清單

**3 種影片風格：**

| 風格 | 適用平台 | 特色 |
|------|---------|------|
| 帶貨口播 | TikTok, YouTube | 主持人介紹、親切感 |
| 商品展示 | Instagram, FB | 360度展示、專業感 |
| 情境故事 | TikTok, Reels | 故事性強、情感共鳴 |

**腳本包含：**
- 每場景的秒數分配
- 畫面描述
- 旁白文案
- 鏡位建議（正面、側面、俯視等）
- 所需道具清單
- 轉場效果（淡入、快切等）
- 配樂風格
- CTA 文案

**範例輸出：**
```
【帶貨口播 - 15秒腳本】

場景 1 (3秒) - 開場
畫面：主持人手持產品
旁白：「嗨大家好！今天要來介紹這個超好用的自拍棒」
鏡位：正面中景

場景 2 (4秒) - 功能展示
畫面：伸縮示範
旁白：「它可以從 20 公分伸長到 100 公分，超方便的！」
鏡位：特寫

...
```

### 4️⃣ 多平台格式轉換

**解決的問題：**
- 不同平台有不同的格式要求
- 手動調整文案格式耗時
- 不熟悉平台規格限制

**功能特色：**
- 自動轉換為平台格式
- 符合字數與規格限制
- 自動整理規格與賣點
- 蝦皮可匯出 CSV

**支援平台：**

**蝦皮 Shopee**
- 標題：60 字限制
- 圖片：1200x1200 px
- 功能：規格列表、賣點整理、CSV 匯出

**Momo 購物**
- 標題：50 字限制
- 圖片：800x800 px
- 功能：分段描述、特色點整理

**轉換內容：**
```
原始文案 →
├─ 蝦皮格式
│  ├─ 標題（60字內）
│  ├─ 描述
│  ├─ 規格列表
│  ├─ 賣點整理
│  └─ CSV 檔案
│
└─ Momo 格式
   ├─ 標題（50字內）
   ├─ 分段描述
   └─ 規格表
```

**CSV 格式範例：**
```csv
商品名稱,商品描述,規格,賣點,價格,庫存,圖片1,圖片2,圖片3
"智能藍牙自拍棒","超好用的自拍神器...","品名：智能藍牙自拍棒 | 材質：鋁合金","✓ 台灣現貨 | ✓ 快速出貨",599,100,"","",""
```

### 5️⃣ FB 廣告素材自動生成

**解決的問題：**
- 不熟悉 FB 廣告規格
- 文案超過字數限制
- 缺乏廣告文案撰寫經驗

**功能特色：**
- 一次產出 1-10 篇廣告
- 自動符合 FB 規格
- 即時格式驗證
- FB 廣告預覽
- 多種文案風格

**FB 廣告規格：**

| 項目 | 限制 | 說明 |
|------|------|------|
| 主要文字 | 建議 125 字內 | 超過會被截斷 |
| 標題 | 40 字以內 | 必須符合 |
| 說明 | 30 字以內 | 必須符合 |
| 圖片 | 1200x628 px | 比例 1.91:1 |
| 圖片文字 | 不超過 20% | 影響投放 |

**廣告類型範例：**

1. **限時特價型**
   ```
   標題：限時特價！智能藍牙自拍棒
   主要文字：🔥 超值優惠來了！智能藍牙自拍棒限時特價中！...
   CTA：立即購買
   ```

2. **品質保證型**
   ```
   標題：智能藍牙自拍棒 - 你的最佳選擇
   主要文字：還在猶豫嗎？千萬別錯過這個改變生活的好物！...
   CTA：了解更多
   ```

3. **熱銷推薦型**
   ```
   標題：熱銷商品 - 智能藍牙自拍棒
   主要文字：🌟 為什麼大家都在買？因為它真的太好用了！...
   CTA：立即選購
   ```

**驗證功能：**
- ✅ 自動檢查字數
- ✅ 標記超過限制的部分
- ✅ 提供修改建議
- ✅ 即時預覽效果

### 6️⃣ 受眾分析

**解決的問題：**
- 不知道產品適合哪些受眾
- 廣告投放目標不明確
- 缺乏市場分析數據

**功能特色：**
- 智能受眾推薦
- 人口統計分析
- 平台建議
- 關鍵字推薦
- 市場規模評估
- 相關性評分

**分析輸入：**
- 方式一：商品名稱
- 方式二：商品網址（AI 自動分析）

**分析輸出：**

**1. 建議受眾群體**
```
受眾 1：旅遊愛好者
- 描述：經常旅行、喜歡記錄旅程的人群
- 市場規模：大眾市場 🚀
- 相關性：95 分
- 建議平台：Instagram, Facebook

受眾 2：社群內容創作者
- 描述：YouTuber、部落客、IG 網紅
- 市場規模：中型市場 📈
- 相關性：90 分
- 建議平台：Instagram, Facebook
```

**2. 人口統計資訊**
- 年齡範圍：18-24, 25-34, 35-44
- 性別分布：女性 65%, 男性 35%
- 興趣標籤：旅遊、攝影、社群媒體、時尚
- 行為特徵：頻繁使用 Instagram、經常線上購物

**3. 關鍵字建議**
```
#自拍 #旅遊 #攝影 #vlog #網美 #打卡 #IG #直播
```

**4. 目標市場**
```
🌏 台灣  🌏 香港  🌏 新加坡  🌏 馬來西亞
```

**應用場景：**
- 設定 FB/IG 廣告受眾
- 規劃內容行銷策略
- 選擇投放平台
- SEO 關鍵字優化
